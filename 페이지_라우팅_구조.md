# 페이지 라우팅 구조

> 기준 코드: `frontend/src/App.tsx`, `frontend/src/main.tsx`  
> 라우팅 방식: `react-router-dom` (`BrowserRouter`, `useLocation`, `useNavigate`)  
> 최종 업데이트: 2026-02-27

---

## 2) 인증 가드/리다이렉트

`App.tsx` 가드 규칙:

- 토큰 없음:
  - 공개 경로(`/login`, `/signup`) 외 접근 시 `/login` 이동
- 토큰 있음:
  - `/`, `/login`, `/signup` 접근 시 `/dashboard` 이동
- 미정의 경로:
  - 토큰 유무에 따라 `/dashboard` 또는 `/login` 이동

---

## 3) Dashboard 탭 동기화

`activeTab`은 URL과 동기화됩니다.

- `/dashboard` -> `dashboard`
- `/dashboard/compare` -> `compare`
- `/dashboard/simulator` -> `simulator`
- `/dashboard/target` -> `target`

탭 변경 시 `navigateTo(view, tab)`가 path로 이동:

```ts
const buildPath = (view: ViewType, tab: TabType): string => {
  if (view === 'profile') return '/profile';
  if (view === 'data-input') return '/data-input';
  if (view === 'reports') return '/reports';
  if (view === 'analytics') return '/analytics';
  if (tab === 'compare') return '/dashboard/compare';
  if (tab === 'simulator') return '/dashboard/simulator';
  if (tab === 'target') return '/dashboard/target';
  return '/dashboard';
};
```

---

## 4) 로그인/회원가입 플로우

1. 로그인 성공 -> 토큰 저장 -> `/welcome` 이동  
2. `WelcomePage` 자동 전환/버튼으로 `/dashboard` 이동  
3. 로그아웃 -> 토큰 제거 -> `/login` 이동

---

## 5) Header/프로필 진입

- 로고 클릭 -> `/dashboard`
- 헤더 탭 클릭 -> `/dashboard/*`
- 프로필 아이콘 -> `/profile`
- 프로필 내부 좌측 메뉴:
  - Dashboard -> `/dashboard`
  - Compare -> `/dashboard/compare`
  - Simulator -> `/dashboard/simulator`
  - Targets -> `/dashboard/target`

---

## 6) Simulator 화면 동작 요약 (`/dashboard/simulator`)

라우팅은 1개지만, 내부는 고정 4단계 섹션으로 동작:

1. `국내 직접 탄소량`: 국내 S1+S2 기반 순노출량/컴플라이언스 비용 계산
2. `전략 배분`: 우선 집행 vs 잔여 이행 비율 시뮬레이션
3. `해외 탄소 배출량`: 해외 S1+S2, EUA, EUR/KRW 환율 결합
4. `시뮬레이션 결과 요약`: 통합 비용과 예산 대비 리스크 등급

입력 상태(`ETS 가격`, `배출 변동`, `무상할당 정책`, `경매 조달 비중`, `연간 탄소 예산`)는 URL 파라미터가 아닌 앱 상태로 관리됩니다.
